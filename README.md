# rotatefile

Just a personal tools written for fun.

## How to run it

```bash
$> git clone https://github.com/pcavezzan/gorotatefile
$> cd rotatefile
$> go build && go install
```

## Options

```bash
$> rotatefile -h
```

## Usage

2 choices:

```bash
$> cat myBigFile.txt | rotatefile
```

or 

```bash
$> rotatefile < myBigFile.txt
```
